# 流程规格：管理终端—校级总览（MVP）

> 适用范围：校级总览页面的数据加载、筛选、统计派生、隐私保护、空状态与失败处理。  
> 本规格不包含个案下钻、导出、后端和真实权限。

---

## 1. 进入流程

```text
用户点击“校级总览”
→ 校验前置共享数据可用
→ 默认统计周期 = 当前学期
→ 默认组织范围 = 全校
→ 统一派生统计 ViewModel
→ 渲染核心指标、重点关注、分布和趋势
```

要求：

- 不使用工作台当前心理老师过滤；
- 不修改共享业务数据；
- 不写业务时间线；
- 不把页面统计结果保存为第二份业务真值。

---

## 2. 数据加载流程

```text
读取学生与学籍
+
读取当前学期测评记录
+
读取共享 warnings
+
读取统一 currentTime
→ 校验数据一致性
→ 排除无法安全统计的异常记录
→ 生成 dataIssues
→ 构建 Overview ViewModel
```

### 2.1 全部加载失败

```text
显示整页失败状态
→ 提供重试
→ 不展示旧统计冒充新结果
```

### 2.2 局部计算失败

```text
其他模块继续展示
→ 失败模块显示局部错误
→ 提供重试或数据核对提示
```

---

## 3. 组织筛选流程

### 3.1 全校

```text
organizationScope = school
→ 统计全部在校学生与全部 warnings
```

### 3.2 年级

```text
选择年级
→ organizationScope = grade
→ 清空班级选择
→ 所有卡片和图表重新派生
```

### 3.3 班级

```text
先选择年级
→ 选择该年级下班级
→ organizationScope = class
→ 所有卡片和图表重新派生
```

不允许：

- 顶部卡片使用全校范围、图表使用年级范围；
- 切换年级后保留旧班级；
- 只更新部分模块。

---

## 4. 当前风险快照流程

```text
过滤当前活动事项
→ 校验 confirmedRiskLevel
→ 仅保留 medium / high / critical
→ 按 studentId 去重
→ 生成当前确认风险学生数与等级分布
```

异常：

- 同一学生多条活动事项；
- 活动预警风险为 low；
- 正式预警及后续状态缺少 confirmedRiskLevel。

异常记录不静默计入正常结果，同时生成 `dataIssues`。

---

## 5. 处置进程流程

当前活动事项统计：

```text
pending_review
observing
formal_warning
in_intervention（显示“待干预”）
pending_retest
referral
```

历史闭环统计：

```text
closed
且闭环时间位于当前学期
```

规则：

- 当前活动状态和已闭环不共用同一百分比分母；
- 预约、改约、取消和待确认不新增主状态；
- 子状态只进入“当前重点关注”。

---

## 6. 当前重点关注派生

### 6.1 待安排干预

```text
currentStatus === formal_warning
且不存在有效干预预约
```

### 6.2 干预预约待确认

```text
currentStatus === in_intervention
且最新有效预约 status === planned
且 currentTime > plannedAt + 60 分钟
```

不自动写 `no_show`。

### 6.3 反馈超时

```text
getEffectiveFeedbackStatus(warning, currentTime) === feedback_overdue
```

### 6.4 复测未完成

```text
currentStatus === pending_retest
且最新计划复测 completedAt 不存在
且 currentTime > plannedAt + 120 分钟
```

### 6.5 复测结果待更新

```text
最新复测 completedAt 存在
且 currentStatus === pending_retest
```

### 6.6 转介中

```text
currentStatus === referral
```

新增跟进后继续计入。

---

## 7. 风险分布流程

### 7.1 年级

```text
按 grade 聚合
→ 在校学生数
→ 当前确认风险学生数
→ 风险占比
→ 中 / 高 / 危险结构
```

### 7.2 班级

```text
按 class 聚合
→ 应用小数量隐私阈值
→ 风险人数 < 3 时显示“少量”
```

排序：

```text
风险占比
→ 危险风险人数
→ 高风险人数
```

---

## 8. 趋势流程

### 8.1 新增确认风险学生

```text
按心理老师首次确认风险的时间归月
→ 同一学生同一月去重
```

### 8.2 正式预警

```text
按正式预警确认事件时间归月
→ 统计事项数
```

### 8.3 闭环

```text
按闭环时间归月
→ 统计事项数
```

### 8.4 转介

```text
按转介发起时间归月
→ 统计事项数
```

当前状态变化不得覆盖历史月份。

---

## 9. 风险线索来源流程

```text
对当前确认风险事项读取 sourceType
→ 聚合 screening_abnormal / ai_chat_trigger / teacher_report
```

禁止：

- 从摘要文本中推断原因；
- 把 AI 风险标记当作心理老师确认风险；
- 将一个学生的异常重复计为多名学生而不提示。

---

## 10. 隐私保护流程

```text
校长 / 心理老师进入总览
→ 只返回聚合 ViewModel
→ 不返回姓名、学号、联系方式和记录正文
```

班级小数量保护：

```text
count < 3
→ exactCount = null
→ displayValue = "少量"
```

不得通过 Tooltip 泄露原始精确值。

---

## 11. 空状态

### 无有效测评

```text
当前统计周期暂无有效测评数据
```

### 有测评、无确认风险

```text
当前暂无心理老师确认的活动风险事项
```

禁止显示：

```text
全校学生均为低风险
```

### 筛选范围无数据

```text
当前范围暂无符合统计口径的数据
```

---

## 12. 页面交互流程

允许：

- 切换组织范围；
- 查看 Tooltip；
- 切换趋势指标；
- 查看统计口径；
- 重试失败模块。

不允许：

- 点击进入学生个案；
- 在总览执行专业操作；
- 修改统计值；
- 写业务时间线。

---

## 13. 响应式与页面布局

桌面端：

```text
核心指标
→ 当前重点关注
→ 两列或三列图表布局
```

窄屏：

```text
模块按优先级纵向排列
```

要求：

- 不出现需要横向拖动整页才能看完的图表；
- 图例和单位始终可见；
- 图表失败不改变其他模块高度；
- 复用最新 App Shell 和侧边栏布局；
- 不重新引入工作台已修复的 `max-width` 大留白或嵌套滚动。

---

## 14. 流程完成条件

1. 所有模块使用同一统计周期和组织范围。
2. 当前快照、历史趋势和子状态压力分开。
3. 页面只读且不写业务时间线。
4. 数据异常被识别并提示。
5. 小数量隐私保护生效。
6. 空状态不会错误暗示心理安全。

## 15. Phase S1.1 派生流程

```text
共享学生 + warnings + 当前时间 + 当前学期
→ 当前确认风险学生按 studentId 去重
→ 生成年级构成（超过 5 个年级时前 4 + 其他）
→ 按正式预警确认事件和真实闭环时间生成月度事项趋势
→ 计算反馈已阅待安排、复测超时未完成及其他重点关注
→ 按 warningId 去重当前阻塞事项
→ 计算闭环率与有效平均闭环周期
→ 输出无身份、无下钻的校长视角 ViewModel
```

反馈已阅仅读取当前反馈轮次的有效记录及细粒度 `psychologistReadAt`，不得回退到兼容字段。复测超时严格使用 120 分钟宽限期。浏览、筛选和图表交互均不写业务时间线。
## Phase S1.2 派生顺序

1. 按组织范围取得在校学生。
2. 按当前学期有效测评记录计算覆盖率，按学生去重。
3. 从活动 warnings 中选取心理老师已确认的中、高、危险风险学生。
4. 分别建立转介、处置积压、协作阻塞事项集合，再以 `warningId` 并集计算当前需关注总数。
5. 将活动事项映射为评估与确认、干预与复测、校外支持三个阶段；将本学期闭环事项作为第四类历史指标。
6. 从当前学期结构化深度测评维度中计算达到关注阈值的学生数，保持量表维度标识，不合并不同量表的同名维度。
7. 按真实确认和闭环时间生成自然月趋势。
8. 按当前年级生成确认风险学生分布。

所有结果由 selector 生成。React 组件不得临时通过 `filter().length` 定义统计口径。
