# 验收矩阵：管理终端—校级总览（MVP）

## 1. 入口与范围

| 编号 | 场景 | 预期结果 |
|---|---|---|
| SO-N001 | 点击侧边栏校级总览 | 展示真实校级总览，不再显示占位页 |
| SO-N002 | 首次进入 | 默认当前学期、全校 |
| SO-N003 | 切换年级 | 所有模块同步更新 |
| SO-N004 | 切换班级 | 所有模块同步更新 |
| SO-N005 | 切换年级 | 旧班级选择被清除 |
| SO-N006 | 浏览与筛选 | 不写预警业务时间线 |
| SO-N007 | 工作台当前老师过滤 | 不影响校级总览全校统计 |

## 2. 测评覆盖

| 编号 | 场景 | 预期结果 |
|---|---|---|
| SO-C001 | 同一学生多次有效测评 | 覆盖人数只计一人 |
| SO-C002 | 未完成测评 | 不计入已完成 |
| SO-C003 | 离校学生 | 不计入当前在校基数 |
| SO-C004 | 完成人数大于在校人数 | 报告数据异常 |
| SO-C005 | 无测评数据 | 显示明确空状态，不先显示 0% |

## 3. 当前风险

| 编号 | 场景 | 预期结果 |
|---|---|---|
| SO-R001 | 活动事项已确认中风险 | 计入当前风险 |
| SO-R002 | 活动事项已确认高风险 | 计入当前风险 |
| SO-R003 | 活动事项已确认危险 | 计入当前风险 |
| SO-R004 | 只有 suggestedRiskLevel | 不计入确认风险 |
| SO-R005 | 活动预警出现 low | 报告数据异常，不作为正常活动风险 |
| SO-R006 | 已闭环事项 | 不计入当前风险 |
| SO-R007 | 同一学生多条活动事项 | 学生人数不重复，报告异常 |
| SO-R008 | 无确认风险 | 显示“当前暂无心理老师确认的活动风险事项” |
| SO-R009 | 无确认风险 | 不显示“全校学生均为低风险” |

## 4. 当前重点关注

| 编号 | 指标 | 验收条件 |
|---|---|---|
| SO-A001 | 待安排干预 | 正式预警且无有效预约 |
| SO-A002 | 干预预约待确认 | 待干预、planned、超过 60 分钟 |
| SO-A003 | 干预预约待确认 | 不自动写 no_show |
| SO-A004 | 反馈超时 | 使用统一有效反馈状态 |
| SO-A005 | 复测未完成 | 待复测、未完成、超过 120 分钟 |
| SO-A006 | 复测结果待更新 | 已完成复测但状态仍待复测 |
| SO-A007 | 转介中 | 状态为转介中 |
| SO-A008 | 新增转介跟进 | 转介中数量不消失 |
| SO-A009 | 单位 | 所有重点关注使用“项” |
| SO-A010 | 交互 | 不提供个案下钻 |

## 5. 主状态分布

| 编号 | 场景 | 预期结果 |
|---|---|---|
| SO-S001 | in_intervention | 页面显示“待干预” |
| SO-S002 | 预约改约 / 未到场 | 不新增主状态 |
| SO-S003 | 当前活动事项 | 统计前六种活动状态 |
| SO-S004 | 已闭环 | 单独统计当前学期闭环事项 |
| SO-S005 | 活动与闭环 | 不使用同一百分比分母 |

## 6. 年级与班级

| 编号 | 场景 | 预期结果 |
|---|---|---|
| SO-O001 | 年级分布 | 展示学生基数、风险人数和占比 |
| SO-O002 | 大年级人数高但占比低 | 不默认排在高占比年级前 |
| SO-O003 | 班级风险人数为 1 或 2 | 显示“少量” |
| SO-O004 | 隐私遮蔽 | Tooltip、ARIA 和数据属性不泄露精确值 |
| SO-O005 | 班级风险人数 >= 3 | 可显示精确数 |
| SO-O006 | 切换组织范围 | 所有卡片和图表口径一致 |

## 7. 趋势

| 编号 | 场景 | 预期结果 |
|---|---|---|
| SO-T001 | 同一学生同月多条事项 | 新增风险学生只计一人 |
| SO-T002 | 正式预警事件 | 按确认时间归月 |
| SO-T003 | 闭环事件 | 按闭环时间归月 |
| SO-T004 | 转介事件 | 按 referredAt 归月 |
| SO-T005 | 当前状态变化 | 不覆盖历史月份 |
| SO-T006 | 切换趋势指标 | 单位同步从“人”切换为“项” |
| SO-T007 | 趋势上升 | 页面不自动输出因果结论 |

## 8. 风险线索来源

| 编号 | 场景 | 预期结果 |
|---|---|---|
| SO-E001 | 普筛异常 | 正确计入 |
| SO-E002 | AI 倾诉触发 | 正确计入 |
| SO-E003 | 班主任上报 | 正确计入 |
| SO-E004 | 自由文本摘要 | 不自动推断风险原因 |
| SO-E005 | 文案 | 明确“来源不等于原因” |

## 9. 页面状态

| 编号 | 场景 | 预期结果 |
|---|---|---|
| SO-P001 | 加载中 | 显示 Skeleton，不先显示 0 |
| SO-P002 | 全部失败 | 显示整页失败和重试 |
| SO-P003 | 单个模块失败 | 其他模块继续展示 |
| SO-P004 | 筛选范围无数据 | 显示范围空状态 |
| SO-P005 | 数据异常 | 显示聚合核对提示 |
| SO-P006 | 数据异常 | 不暴露学生身份 |
