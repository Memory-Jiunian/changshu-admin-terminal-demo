# 工作台流程规格

共享 warnings → 当前老师与活动事项准入 → 六类待办和今日提醒派生 → 互斥与排序 → 页面展示。

任务点击 → 保存 `WorkbenchReturnContext` → 打开正确预警详情 → 等待目标锚点渲染 → 定位并短暂高亮。

新反馈任务只负责打开正确事项并定位反馈模块。反馈模块渲染后保持未读；心理老师明确点击“标记为已查看”时，统一 action 写入未读反馈记录的 `psychologistReadAt` 并同步兼容字段。该阅读更新不写时间线。

工作台新反馈来源且仍有未读反馈时，关闭详情先确认：继续查看、关闭并保留待办，或标记已查看并关闭。普通预警入口不启用关闭保护。

返回工作台 → 恢复筛选和滚动 → 重新派生；已失效任务不恢复，其他任务保留。

目标模块不存在时打开最新事项概况并提示状态已更新；warningId 不存在时保留工作台并提示事项不存在。
