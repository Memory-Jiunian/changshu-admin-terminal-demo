# 系统设置 MVP 页面级 PRD

## 页面结构

```text
系统设置
├── 权限说明
├── 学校信息
├── 组织架构
├── 教师管理
├── 学生管理
└── 批量导入
```

顶部明确说明“当前演示账号具备资料维护权限”。页面复用 App Shell，使用页面级纵向滚动、分页表格和统一安全高度 Dialog，不产生整页横向滚动。

## 学校信息

字段：学校名称、学校编码、当前学年、当前学期、学期开始日期、学期结束日期。默认只读，编辑后提供保存和取消。学校名称同步 Topbar；学期范围同步校级总览。

## 组织架构

使用年级 / 班级列表与选中详情。支持新增、编辑、排序、指定班主任和停用。年级名称校内唯一；班级名称在同年级唯一；班主任只能选择启用教师。

## 教师管理

列表展示姓名、工号、角色、负责范围、手机号、状态和操作。支持搜索、筛选、新增、编辑和停用。停用前检查活动预警负责人、当前班主任和未来预约负责人。

## 学生管理

列表展示姓名、学号、年级、班级、班主任、学籍状态和操作。支持搜索、筛选、新增、编辑、转班 / 升年级、毕业和离校。学生详情仅展示基础资料和学籍历史，不展示心理档案内容。

## 批量导入

流程为数据类型、导入模式、上传、校验预览、确认和结果。预览展示总行数及新增、更新、跳过、警告、错误数量，并列出行号、唯一键、名称、处理类型、差异和校验信息。

## 页面状态

覆盖加载、空数据、保存中、保存失败、引用冲突、解析失败、格式错误、预览过期、原子提交失败、保存成功和未保存离开保护。
