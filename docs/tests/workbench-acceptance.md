# 工作台 MVP 验收

- [x] 只展示当前心理老师的活动事项
- [x] 六类待办和一类提醒按规则派生
- [x] 今日复测提醒不计入待办总数
- [x] 新反馈与反馈超时互斥
- [x] 今日复测提醒与复测结果待更新互斥
- [x] 转介跟进中持续存在且不标逾期
- [x] 稳定 ID、排序和数据异常可测试
- [x] 六个预警详情锚点存在并可定位
- [x] 目标模块缺失安全回退概况
- [x] 新反馈模块展示后保持未读，明确确认后才标记已读
- [x] 已读更新不改变反馈记录和业务时间线
- [x] 返回上下文恢复筛选、滚动和选中任务
- [x] 首次进入展示真实工作台
- [x] 未实现页面展示占位而非预警管理
- [x] 预警管理和学生档案回归通过

## Phase W1.1

- [x] 共享 mock 覆盖六类主动待办和一类今日提醒
- [x] 今日提醒显示“查看安排”，定位复测模块且只查看不消失
- [x] 任一反馈记录未读即派生有新反馈
- [x] 工作台和普通预警入口共用“标记为已查看”
- [x] 标记已读保留反馈正文、反馈记录和业务时间线
- [x] 工作台新反馈来源未读关闭时出现阻断式保护 Dialog
- [x] 关闭并保留待办不修改阅读状态
- [x] 标记已查看并关闭后任务消失
- [x] 普通预警入口和 warningId 不一致时不触发关闭保护
