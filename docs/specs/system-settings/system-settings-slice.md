# 系统设置 MVP 功能切片

## 目标

系统设置维护全系统共享的学校、组织、教师和学生基础资料，不是普通偏好设置页。当前 Demo 由具备资料维护权限的心理老师操作；真实产品应由学校资料管理员或被授权心理老师操作。

页面包含：

1. 学校信息
2. 组织架构
3. 教师管理
4. 学生管理
5. 批量导入

## 不变量

- 学生档案继续为只读聚合页，不承担基础资料编辑。
- 学生 `studentId`、教师 `teacherId` 稳定，学号和工号分别为业务唯一键。
- 班级属于一个年级，最多有一个当前班主任。
- 学籍变化保留历史区间。
- 有业务引用的数据不硬删除；教师、年级和班级使用停用或归档，学生使用毕业或离校。
- 当前资料更新不得覆盖历史预警、反馈、干预、复测、转介和时间线快照。
- 系统设置写独立管理操作记录，不写预警处置时间线。

## 导入

固定模板支持组织、教师和学生的 `.xlsx` / `.csv` 导入，模式为“仅新增”或“新增并更新”。确认前必须生成 create、update、skip、warning、error 和字段差异预览。存在 Error 时禁止确认；提交在内存副本执行，全部成功后一次替换，失败整批回滚。

## 不实现

真实权限、密码安全、批量导出、灵活列映射、多学校、后端、第三方学籍同步、异步队列和永久删除。
