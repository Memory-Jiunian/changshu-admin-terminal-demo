# 工作台 MVP 验收

- [x] 只展示当前心理老师的活动事项
- [x] 六类待办和一类提醒按规则派生
- [x] 今日复测提醒不计入待办总数
- [x] 新反馈与反馈超时互斥
- [x] 今日复测提醒与复测结果待更新互斥
- [x] 转介跟进中持续存在且不标逾期
- [x] 稳定 ID、排序和数据异常可测试
- [x] 六个预警详情锚点存在并可定位
- [x] 目标模块缺失安全回退概况
- [x] 新反馈只在正确反馈模块展示后标记已读
- [x] 已读更新不改变反馈记录和业务时间线
- [x] 返回上下文恢复筛选、滚动和选中任务
- [x] 首次进入展示真实工作台
- [x] 未实现页面展示占位而非预警管理
- [x] 预警管理和学生档案回归通过
