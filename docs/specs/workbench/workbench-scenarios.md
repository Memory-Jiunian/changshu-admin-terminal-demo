# 工作台场景化走查

1. 首次进入管理终端，默认显示工作台及当前老师的任务概览。
2. 点击概览类型只筛选当前待办，不影响今日提醒。
3. 点击待办打开正确 warningId，并定位风险依据、反馈、复测、转介或操作区。
4. 返回工作台恢复筛选、滚动和选中任务，再根据最新 warnings 重算。
5. 同一学生可以出现不同类型任务，不合并。
6. 新反馈正确展示后仍保留任务；心理老师点击“标记为已查看”后任务才消失。
7. 转介跟进新增后任务保留，安排复测离开转介后消失。
8. 无待办时显示明确空状态，不用趋势或虚构任务填充。
9. 工作台新反馈详情仍未读时关闭，必须选择继续查看、关闭并保留待办或标记已查看并关闭。
10. 普通预警入口也可明确标记已查看，并同步影响工作台任务。
